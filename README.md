## Installation

### 1. Install package

```shell
$ pip3 install vedro-dev
```

### 2. Enable plugin

```python
# ./vedro.cfg.py
import vedro
import vedro_dev

class Config(vedro.Config):

    class Plugins(vedro.Config.Plugins):

        class VedroDev(vedro_dev.VedroDev):
            enabled = True

```
